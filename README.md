# StoryCrafter – AI Story Generator for Kids

StoryCrafter is an AI-powered web application that generates creative short stories for children based on user input. It uses Natural Language Processing with the GPT-2 model to create engaging and imaginative stories instantly.

# 🚀 Features

✨ Generate unique stories instantly

🎯 Custom inputs (Theme, Character, Location)

🌐 Simple web interface using Gradio

⚡ Runs locally (No API required)

🧠 Powered by AI

# 🧠 Technologies Used

💻 Python

🤖 Hugging Face Transformers

🧠 GPT-2

⚙️ PyTorch

🌐 Gradio

# 🧩 How It Works

User enters:

Theme

Character

Location

The system creates a prompt

The GPT-2 model generates a story

Output is displayed using Gradio

# 🌐 Does It Use an API?

❌ No, this project does NOT use any API.

✔ The model is downloaded once using Hugging Face Transformers
✔ Runs completely on your local system
